ClinicFlow™ – Booking System

Thank you for your purchase.

To get started:

1. Upload the /site folder to Cloudflare Pages
2. Deploy the Worker from /worker/api.js
3. Create your database using /database/schema.sql
4. Follow the setup guide in /docs

This system allows you to:

* Take bookings online
* Accept deposits via Stripe
* Manage appointments
* Reduce no-shows

For full setup instructions, open the setup guide.
